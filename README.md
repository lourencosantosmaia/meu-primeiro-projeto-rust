# 🍎 Sistema de Gestão Escolar em Rust

Este é um projeto desenvolvido para automatizar o lançamento de notas e o cálculo de desempenho de turmas escolares. Desenvolvido em **Rust**, o sistema foca em performance, segurança de memória e facilidade de uso.

## 🚀 Funcionalidades
- **Lançamento de Notas:** Registro interativo de nomes e notas.
- **Persistência de Dados:** Todos os dados são salvos automaticamente em um arquivo `.txt`.
- **Relatórios:** Visualização rápida de todos os alunos registrados e seus status (Aprovado/Reprovado).
- **Estatísticas:** Cálculo automático da média geral da turma.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Rust
- **IDE:** Visual Studio Code
- **Armazenamento:** File System (TXT)
